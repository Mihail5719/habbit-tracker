# 🎯 Habbit App — Трекер привычек

**Habbit App** — это современное веб-приложение для отслеживания ежедневных привычек, формирования полезных навыков и анализа личного прогресса.

Приложение работает полностью в браузере, не требует серверной части и сохраняет все данные локально на устройстве пользователя.

## ✨ Возможности

- ✅ **Управление привычками**: создание, редактирование и удаление привычек.
- 🗂️ **Категории**: 5 категорий (Спорт, Питание, Здоровье, Обучение, Другое) с цветовой индикацией.
- 🔍 **Фильтрация**: быстрый фильтр привычек по категориям в боковом меню.
- 📊 **Статистика**: подробная статистика по всем привычкам и общий прогресс.
- 📝 **Дневник**: добавление и редактирование комментариев к каждому пройденному дню.
- 🌙 **Тёмная тема**: переключение между светлой и тёмной темой с сохранением выбора.
- 💾 **Экспорт и Импорт**: возможность скачать резервную копию данных в JSON и загрузить её обратно.
- 📱 **Адаптивность**: корректное отображение на десктопах, планшетах и мобильных устройствах.

## 📱 Адаптивность

Проект оптимизирован для десктопа (экраны от 1024px).

### Поддерживаемые устройства:
- ✅ **Десктоп** (1920px+) — полный функционал, меню слева
- ✅ **Планшеты** (768px - 1024px) — меню снизу, удобная навигация
- ✅ **Большие телефоны** (480px+) — базовая адаптивность, иконки без текста
- ⚠️ **Маленькие телефоны** (360px × 640px) — имеет ограничения:
  - Иконки привычек требуют горизонтальной прокрутки
  - Навигация по меню может быть неудобной на сенсорном экране
  - Рекомендуется использовать на планшетах или больших телефонах

### Технические детали:
- Реализованы 3 брейкпоинта: **1024px**, **768px**, **480px**
- На мобильных устройствах меню перемещается вниз экрана
- Текстовые метки кнопок скрываются на экранах до 480px
- Все иконки заменены на SVG для одинакового отображения на всех устройствах

## 🛠️ Технологии

Проект написан на чистом (Vanilla) JavaScript без использования сторонних библиотек и фреймворков.

- **HTML5** — семантическая разметка.
- **CSS3** — CSS-переменные (кастомные свойства), Flexbox, CSS Grid, медиа-запросы для адаптивности.
- **JavaScript (ES6+)** — работа с DOM, событиями, localStorage, FileReader API, Blob API.

## 🚀 Как запустить

Поскольку приложение не требует бэкенда, запустить его очень просто:

### Способ 1: Открыть в браузере
1. Скачайте или клонируйте репозиторий.
2. Откройте файл index.html в любом современном браузере (Chrome, Firefox, Safari, Edge).

### Способ 2: Через Live Server (рекомендуется)
1. Установите расширение **Live Server** в VS Code.
2. Откройте папку проекта в VS Code.
3. Нажмите правой кнопкой мыши на index.html и выберите **Open with Live Server**.

## 💡 Как это работает

- **Хранение данных**: Все привычки и настройки сохраняются в localStorage браузера. Данные не отправляются на сервер.
- **Архитектура**: Код разделен на логические блоки (утилиты, рендеринг, обработка событий, инициализация).
- **Темы**: Смена темы реализована через CSS-переменные. При переключении JS добавляет класс .dark-theme к body.
